# encrypt-ayx-guide

本仓库用于归档和发布多个独立 HTML 页面。这些页面以静态方式呈现，不针对任何特定域名或网站，仅作为通用内容存档与展示用途。

## 项目简介

- 仓库名称：encrypt-ayx-guide  
- 用途：存放并发布多个独立 HTML 页面  
- 特性：纯静态、无后端依赖、可直接通过 GitHub Pages 访问每个页面  
- 目标：提供一种简洁、轻量的方式归档和展示 HTML 内容，方便查阅与分享

## 目录说明

```
.
├── index.html          # 主页面（可选）
├── page-1.html         # 示例页面 1
├── page-2.html         # 示例页面 2
└── ...
```

- 每个 `.html` 文件均为一个独立的页面，互不依赖。
- 文件名建议使用有意义的英文或拼音，便于记忆和引用。
- 目录结构可根据需要调整，但保持扁平化更易维护。

## 页面归档说明

- 所有页面均为纯 HTML，不含服务器端脚本。
- 页面内可包含 CSS、JavaScript 等客户端资源，但请尽量内联或使用相对路径引用外部资源。
- 页面内容应保持稳定，不建议频繁修改已归档的页面。
- 如需新增页面，直接添加新 `.html` 文件并提交即可。

## 维护说明

- 本仓库欢迎提交改进或新增页面，但请确保内容不违反 GitHub 服务条款。
- 提交前请检查页面在不同浏览器下的基本兼容性。
- 不建议在页面中嵌入跟踪脚本、广告或任何形式的动态外部请求。
- 如需删除某个页面，请直接删除对应文件并更新本 README（如适用）。
- 仓库维护者保留对页面内容进行审核或调整的权利。

## 使用方式

1. 克隆或下载本仓库。
2. 在本地打开任意 `.html` 文件即可查看。
3. 若启用 GitHub Pages，可通过 `https://<用户名>.github.io/encrypt-ayx-guide/` 访问主页面，或通过 `https://<用户名>.github.io/encrypt-ayx-guide/page-1.html` 访问具体页面。
